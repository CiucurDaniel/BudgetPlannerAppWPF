# BudgetPlanner
WPF Windows application used to plan your monthly budget.

# Development team:

Opra Andrei
Ciucur Daniel

# Functionality

* Set monthly budgets for different types of categories (bills, groceries, economies, ...)
* Add you expenses
* View all your expenses
* Print expenses on PDF or TXT (optionally)
* Currency convertor
